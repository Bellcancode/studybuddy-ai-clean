# studybuddy-ai

StudyBuddy AI is an AI-powered learning assistant built with React, FastAPI, and Google's Gemini API.

## Features

- 📄 Upload PDF Notes
- 🤖 AI Chat Assistant
- 📝 AI Summary Generator
- ❓ AI Quiz Generator
- 📅 AI Study Planner

## Technologies Used

- React
- Tailwind CSS
- FastAPI
- Python
- Gemini API
- PyMuPDF

## Installation

### Backend

```bash
cd backend

python -m venv .venv

pip install -r requirements.txt

uvicorn main:app --reload
```

### Frontend

```bash
cd frontend

npm install

npm run dev
```
